# Linux: add Icon field to .desktop file and install PNG icon

The `worktree-runner.desktop` file written by `src/scheme/linux.rs` has no `Icon=` field.

Source: `worktree-io/landing/public/favicon.svg` → export PNG at 256×256.

Steps:
1. Install PNG to `~/.local/share/icons/hicolor/256x256/apps/worktree-runner.png` during `worktree scheme install`.
2. Add `Icon=worktree-runner` to the `[Desktop Entry]` block.
3. Bundle the PNG via `include_bytes!` or ship as a release asset.
4. Update `uninstall()` to remove the icon file.

Depends on #22.
