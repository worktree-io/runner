# Pre-commit: enforce `cargo fmt` formatting

Run `cargo fmt --check` in the pre-commit hook to prevent unformatted code from being committed.

## Requirements

* Add a `cargo fmt --check` step to the pre-commit hook
* Fail the commit if any files are not formatted
* Print the names of unformatted files and instruct the author to run `cargo fmt`
