# Add version command

## Summary

Add a `version` subcommand and/or `-V`/`--version` flag that prints the current crate version to stdout.

## Implementation

1. Add a `version` subcommand to the CLI definition
2. Print the version using `env!("CARGO_PKG_VERSION")`
3. Ensure `-V` / `--version` short flag works (clap handles this automatically when configured)

## Acceptance Criteria

- [ ] `runner version` prints the current version (e.g. `0.6.1`)
- [ ] `runner --version` / `runner -V` also works
- [ ] Version is sourced from `CARGO_PKG_VERSION` (no hardcoding)
