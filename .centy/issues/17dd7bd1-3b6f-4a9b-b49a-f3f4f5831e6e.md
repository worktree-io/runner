# runner config set editor <name> should work (not just editor.command)

## Problem

Running \`runner config set editor tmux\` fails with:

```
Error: Unknown config key: editor
```

The config key for editor is \`editor.command\`, but users naturally try \`editor\` since the CLI help and README refer to the editor setting generically.

## Expected Behavior

Either:
1. Accept \`editor\` as an alias for \`editor.command\` in \`set_value\` / \`get_value\`, or
2. Improve the error message to say: *"Unknown config key: editor — did you mean editor.command?"*

## Steps to Reproduce

```bash
runner config set editor tmux
# Error: Unknown config key: editor
```

Correct command is:

```bash
runner config set editor.command tmux
```
